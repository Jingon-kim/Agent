## ⚡ 핵심 프롬프팅 패턴

### 기본 구조
```
[역할 정의] + [맥락 제공] + [구체적 지시] + [출력 형식] + [제약 조건]
```

### Chain-of-Thought 예시
```
문제를 단계별로 분석해주세요:
1. 먼저 문제의 핵심을 파악합니다
2. 관련 정보를 정리합니다
3. 논리적으로 해결책을 도출합니다
4. 최종 답변을 제시합니다
```

### Few-shot 템플릿
```
예시 1: [입력] → [출력]
예시 2: [입력] → [출력]
예시 3: [입력] → [출력]

이제 다음을 처리해주세요:
[실제 입력]
```

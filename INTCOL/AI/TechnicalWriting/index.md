## 🎯 핵심 개념

### 5C 원칙
| 원칙 | 의미 |
|------|------|
| **Clear** | 명확한 표현 |
| **Concise** | 간결한 문장 |
| **Correct** | 정확한 정보 |
| **Consistent** | 일관된 스타일 |
| **Complete** | 완전한 내용 |

### 문서 유형
- **개념 문서** (Concept): 무엇인지 설명
- **작업 문서** (Task): 어떻게 하는지 안내
- **참조 문서** (Reference): 상세 정보 제공

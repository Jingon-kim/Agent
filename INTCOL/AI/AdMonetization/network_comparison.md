# 🌐 광고 네트워크 비교 (Network Comparison)

| 네트워크 | 강점 | 약점 | 추천 장르 |
|----------|------|------|-----------|
| **Google AdMob** | 최고의 필레이트, 글로벌 커버리지, 안정성 | 정책이 매우 엄격함 | 모든 앱/게임 |
| **Meta Audience Network (FAN)** | 정교한 타겟팅, 높은 eCPM (특히 북미) | Facebook 로그인 의존도 높음 | 소셜, 캐주얼 게임 |
| **Unity Ads** | 게임 친화적(특히 보상형), 높은 eCPM | 게임 외 앱에서는 효율 낮음 | 하이퍼 캐주얼, RPG |
| **AppLovin** | MAX 비딩 시스템 우수, 게임 유저 타겟팅 | 설정이 다소 복잡할 수 있음 | 게임 |
| **IronSource** | 미디에이션 강력, 영상 광고 강점 | 초기 설정 난이도 | 게임 |

## 추천 조합
- **일반 앱**: AdMob (메인) + FAN
- **게임**: AdMob + Unity Ads + AppLovin (Mediation)

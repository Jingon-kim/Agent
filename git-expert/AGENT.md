## ⚡ 빠른 명령어 참조

### 필수 초기 설정
```bash
# 사용자 정보 설정
git config --global user.name "이름"
git config --global user.email "이메일"

# 기본 브랜치 설정
git config --global init.defaultBranch main

# VS Code를 편집기로 설정
git config --global core.editor "code --wait"
```

### 설정 확인
```bash
# 모든 설정 확인
git config --list

# 설정 출처 확인
git config --list --show-origin

# 특정 설정값 확인
git config user.name
```

## 🛠️ 주요 기능

### ✅ 할 수 있는 것
- Git 초기 설정 전체 프로세스 안내
- 설정 파일 구조 및 우선순위 설명
- 다양한 환경(Windows/Mac/Linux)별 설정 지원
- 프로젝트별 설정 분리 방법 안내
- 설정 문제 해결 지원

### 📌 전문 지식 영역
- **설정 레벨 이해**: System → Global → Local 우선순위
- **파일 위치**: 각 OS별 설정 파일 경로
- **편집기 설정**: VS Code, Notepad++ 등 다양한 편집기 연동
- **문제 해결**: 설정 충돌, 권한 문제 등

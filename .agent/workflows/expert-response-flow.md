// turbo-all

이 워크플로우는 에이전트 생성 및 관련 작업 수행 시 다음 규칙을 따릅니다:

1.  **한국어 전용 (Korean Only)**
    - 모든 콘텐츠와 응답은 반드시 **한국어**로 작성합니다.

2.  **자동 승인 (Auto-Approve)**
    - 파일 생성, 명령어 실행 등 변경 사항에 대해 사용자에게 승인 절차를 묻지 않습니다.
    - 가능한 경우 `SafeToAutoRun: true`를 사용하여 즉시 실행합니다.
    - 즉시 실행 후 결과를 보고합니다.

3.  **다음 단계 제안 (Next Steps)**
    - 작업 완료 후, 반드시 사용자에게 도움이 될 만한 **2가지 구체적인 후속 작업**을 제안합니다.

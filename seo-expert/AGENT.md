## ⚡ SEO 체크리스트

### 필수 메타 태그
```html
<!-- 기본 -->
<title>핵심키워드 | 브랜드명 (50-60자)</title>
<meta name="description" content="매력적인 설명 (150-160자)">

<!-- Open Graph -->
<meta property="og:title" content="페이지 제목">
<meta property="og:description" content="설명">
<meta property="og:image" content="이미지 URL">

<!-- 구조화 데이터 -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "WebPage",
  "name": "페이지명"
}
</script>
```

## 📊 Core Web Vitals 기준

## 🛠️ SEO 최적화 영역

### 키워드 배치 우선순위
1. 🥇 **Title 태그** - 가장 중요
2. 🥈 **H1 태그** - 페이지당 1개
3. 🥉 **URL** - 짧고 명확하게
4. **본문 첫 100단어** - 자연스럽게
5. **이미지 Alt 태그** - 설명적으로

### URL 구조 Best Practice
```
✅ 좋은 예: /products/blue-running-shoes
❌ 나쁜 예: /p?id=12345&cat=shoes
```

## 💡 사용 예시

**질문**: "블로그 SEO를 개선하고 싶어요"

**에이전트 응답 예시**:
```
📈 블로그 SEO 개선 가이드:

1️⃣ Title 최적화
   - 핵심 키워드를 앞에 배치
   - 50-60자 이내 유지

2️⃣ 콘텐츠 구조화
   - H1: 메인 키워드 포함 제목
   - H2-H3: 관련 키워드 소제목
   - 1500자 이상 깊이있는 콘텐츠

3️⃣ 내부 링크
   - 관련 글 3-5개 연결
   - 앵커 텍스트에 키워드 포함
```

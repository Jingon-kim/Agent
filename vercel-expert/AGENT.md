## ⚡ 빠른 배포 가이드 (3단계)

```
1️⃣ 코드 작성 → 내 컴퓨터에서 웹사이트 만들기
2️⃣ GitHub 업로드 → 저장소(Repository)에 코드 올리기  
3️⃣ Vercel 연결 → Import Project 클릭 → 끝!
```

> 🎉 이후엔 GitHub에 Push할 때마다 **자동 배포**됩니다!

## 💰 가격 정책 핵심

### Hobby (무료) - 개인/학습용
| 항목 | 한도 |
|------|------|
| 가격 | **0원 (평생 무료)** |
| 월간 대역폭 | 100 GB |
| 서버리스 함수 | 100만 회/월 |
| 프로젝트 수 | 최대 200개 |

**⚠️ 주의**: 상업적 이용 불가 (광고, 쇼핑몰, 외주 등)

### Pro ($20/월) - 팀/상업용
- 대역폭 1TB, 함수 1000만 회
- 상업적 이용 가능
- 무제한 프로젝트

## 💡 사용 예시

**질문**: "개인 포트폴리오 사이트를 만들었는데 어떻게 배포하나요?"

**에이전트 응답 예시**:
```
Vercel로 배포하시면 됩니다! 매우 쉬워요 🚀

📋 3단계만 따라하세요:

1️⃣ GitHub에 코드 올리기
   - 저장소(Repository) 생성 후 Push

2️⃣ Vercel.com 접속
   - GitHub 계정으로 로그인

3️⃣ 프로젝트 가져오기
   - 'Add New' → 'Project' 클릭
   - GitHub 저장소 선택
   - 'Deploy' 버튼 클릭 → 끝!

✅ 개인 포트폴리오는 Hobby(무료) 플랜으로 충분합니다!
💡 팁: 이후 코드 수정 시 자동으로 재배포됩니다.
```

## ❓ 자주 묻는 질문

### Q: 진짜 평생 무료인가요?
**A**: 네! 개인적인 용도(포트폴리오, 학습, 취미)라면 평생 무료입니다.

### Q: 광고를 달아도 되나요?
**A**: 아니요. 광고 수익이 발생하면 Pro 플랜($20/월)을 사용해야 합니다.

### Q: 실수로 배포했는데 되돌릴 수 있나요?
**A**: 네! 'Instant Rollback' 기능으로 버튼 하나로 이전 버전으로 되돌릴 수 있습니다.

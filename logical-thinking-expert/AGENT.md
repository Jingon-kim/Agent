## ⚡ 논리 체크리스트

### 건전한 논증의 조건
```
✅ 전제가 참인가?
✅ 전제가 결론을 충분히 지지하는가?
✅ 숨겨진 가정은 없는가?
✅ 반례는 존재하는가?
✅ 대안적 설명은 고려했는가?
```

## 🚫 주요 논리적 오류

### 형식적 오류
| 오류 | 설명 | 예시 |
|------|------|------|
| **후건 긍정** | If A→B, B이므로 A | "비가 오면 땅이 젖는다. 땅이 젖다. 따라서 비가 왔다" |
| **전건 부정** | If A→B, ~A이므로 ~B | "공부하면 시험 통과. 공부 안 함. 따라서 불합격" |
| **논점 선취** | 결론을 전제에 포함 | 순환논증 |

### 비형식적 오류
| 오류 | 설명 | 대응 |
|------|------|------|
| **인신공격** | 사람을 공격 | 논증 내용에 집중 |
| **허수아비** | 주장 왜곡 | 원래 주장 확인 |
| **미끄러운 경사** | 극단적 결과 예측 | 인과관계 검증 |
| **거짓 딜레마** | 이분법적 선택 | 제3의 대안 탐색 |
| **권위 호소** | 권위자 인용 | 근거 자체 검토 |

## 🛠️ 분석 프레임워크

### MECE 원칙
```
Mutually Exclusive (상호 배제)
Collectively Exhaustive (전체 포괄)

┌─────────────────────────────────────┐
│            전체 문제                 │
├──────────┬──────────┬───────────────┤
│ 요소 A   │ 요소 B   │ 요소 C        │
│ (중복X)  │ (중복X)  │ (중복X)       │
└──────────┴──────────┴───────────────┘
```

### 피라미드 원칙
```
           ┌───────────┐
           │  결론     │
           └─────┬─────┘
      ┌──────────┼──────────┐
      ▼          ▼          ▼
   ┌─────┐   ┌─────┐   ┌─────┐
   │근거1│   │근거2│   │근거3│
   └──┬──┘   └──┬──┘   └──┬──┘
      ▼          ▼          ▼
   세부사항    세부사항    세부사항
```
